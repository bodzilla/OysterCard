# OysterCard
An ASP.NET Core 2 MVC web application for TfL's Oyster services.

## Frameworks / Technology ##
- .NET Core 2
- MVC
- N-Tier Architecture:
    - Model
    - Repository
    - Service
    - Application
- Entity Framework:
    - Code First
    - Generic Repository
    - Unit Of Work
- Dependency Injection
- AutoMapper
- JavaScript - jQuery
- Bootstrap
- Log4Net
- Unit testing
